# Clios

Collection of my projects in one program

NotYetAnother (^>w<^) project

![Clios](https://memoridevel.ru/clios.png)

***

## Idea

* prototype of operating system
* all console projects in one program

## Features

* text-based interface
* True Color support
* use of pseudographics

## Details

* storing encrypted account data in a file
* different levels of access depending on the type of account
* settings
* personalization settings
* drawing with symbols of different colors and transparency
* ability to load/save the drawing in paper format
* games and rating

***

## Build

```bash
chmod +x build.sh

./build.sh Debug Linux # for Debug on Linux
./build.sh Release Linux # for Release on Linux
./build.sh Debug Win32 # for Debug on Windows
./build.sh Release Win32 # for Release on Windows
```
