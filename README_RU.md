# Clios

Коллекция моих проектов в одной программе

![Clios](https://memoridevel.ru/clios.png)

[ [English](README.md) | Русский ]

## Идея
* прототип операционной системы
* все консольные проекты в одной программе

## Особенности
* текстовый интерфейс
* поддержка True Color
* использование псевдографики

## Подробности
* хранение зашифрованных данных учётных записей в файле
* различный уровень доступа в зависимости от типа учётной записи
* панель управления
* настройки персонализации
* рисование символами разных цветов и прозрачности
* возможность загрузить/сохранить рисунок в формате Paper
* игры и рейтинг
